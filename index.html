# 介绍

一句话介绍：滴普科技是全场景数据智能服务商。

公司基于云原生互联网框架，综合5G、IoT、大数据、AI、云计算等新技术，形成可高度扩展的商业智能和产业智能的平台产品，为组织提供全场景数据智能服务。

官网：https://cloud.deepexi.com/

公司正值稳定上升期，目前需要这样的人才：

  • 精益求精，对技术有品味，有执著追求，热爱开源的人, 一起来做开源贡献;
  • 工程能力突出，业务梳理能力强，能把复杂的逻辑理顺，让事情井然有序，在业务中独当一面, 在实践中解决问题的人;
  • 坚持写作的人，因为我们经常写邮件、写文档、写计划、写博客，我希望我的团队成员能够写。

总结就是：如果只会写代码，那不适合我们。

工作地点：广州，成都，南京，北京

媒体报道：

  • 滴普科技完成3500万美元A轮融资
  • 进军企业数字化领域，滴普科技半年内累计完成近亿元融资

# 要求

一句话概括：我们需要的不仅仅是会写业务代码的人。你至少要能在业务中积累沉淀，有所产出；你要维护开源项目；还要能在知乎专栏输出文章。如果你不给自己设限，想做全栈，那就更好了。

我们的技术栈：

  • @femessage/create-nuxt-app
  • qiankun
  • nuxt-micro-frontend
  • vue-sfc-cli
  • 我们自己的组件
  • Jest
  • PWA
  • Verdaccio
  • Yarn
  • Cypress
  • Travis CI
  • Lighthouse
  • Netlify
  • Aliyun OSS
  • Gitlab
  • Yapi
  • Nestjs
  • Egg
  • Flutter

不过技术只是工具，生活才是真正的追求。

我们希望你是能真正动手解决实际问题的人。

同时，我们希望你有自己的兴趣爱好， 一起把事情做好，还能一起玩。

我们相信热爱生活、努力解决问题，让生活更美好的人，技术不会差到哪里去。

以下是扣分项：

  • 不会Git，不知道如何解决冲突
  • 无法翻墙，认为百度就够了
  • 编码规范什么的不重要，耽误时间
  • 文档不重要；没时间写，也不想写
  • 只管开发, 不管测试

以下是加分项(符合的越多，就越欢迎)：

  • 熟练在 Mac OS X / Linux 上开发
  • 坚持写技术博客（或微信公众号），文章超过 10 篇
  • Github 有前端项目超过 200 star，或给大型项目贡献过代码
  • 英文了得，可以写英文文章
  • 可以在 stackoverflow 投赞成票
  • 有成熟上线的产品
  • 发布过npm模块，并且已被投入生产使用
  • 对新事物敏感，喜欢黑科技
  • 做的了设计，搞的了运维，你不仅仅是个前端
  • 风格细腻，注重细节
  • 对美（程序、设计）有追求
  • 乐于分享
